Prototype

Prototype is a small Windows program written in C++ with minimal external libraries (Win32 API, Direct3D, stb_truetype, stb_image). Eventually, it will become a small engine for an asteroids clone. To talk about it in any depth would require a blog and I am not sure if this is quite the right setting for a blog.

Some of the current features include:

  • 2, 3, and 4D vectors and 4x4 matrices with floating point units
    • Some support for SIMD on 4D vectors (add, subtract, Hadamard product, scaling, and broadcasting)
    • Standard matrix operations (multiplication, addition, subtraction, transpose)
    • Perspective, LookAt, Transformations, Translation, and Rotations about X, Y, Z axes
    • Triangle math (Winding order, transformations under matrices)
  • Textures
    • Alpha blending
  • Basic Font Rendering
    • Single line rendering
  • Basic Input
    • Keyboard input
  • Window Creation
  • Basic File I/O
    • Blocking-based API
  • Basic Debugging Facilities
    • Assert
    • Custom assertions for Direct3D API
  • Image Decoding for QOI images
